Autonomous Navigation & Control Stack

A modular autonomy architecture that cleanly separates estimation, planning, smoothing, sampling, and control into independently tunable subsystems.

Overview

This project formalized subsystem contracts so trajectory generation and tracking could evolve without destabilizing state estimation or hardware interfaces.

What I Built

  • Defined module boundaries for estimator, planner, smoother, sampler, and controller stages.
  • Implemented parameter structs to decouple tuning values from algorithmic logic.
  • Standardized data handoff formats for pose, path, and control references.
  • Added deterministic scheduling assumptions for predictable end-to-end latency.
  • Introduced clear ownership of coordinate transforms and unit conventions.
