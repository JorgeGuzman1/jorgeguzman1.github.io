Simulation & Verification Framework

Offline framework used to validate estimation, planning, and control behavior under noisy, non-ideal scenarios before running on hardware.

Overview

This framework provides repeatable stress scenarios to expose integration faults, coordinate-frame mistakes, and tuning fragility early in the development loop.

What I Built

  • Built offline simulation harness for fast autonomy iteration without hardware dependency.
  • Injected stochastic sensor noise to test estimator and controller resilience.
  • Added parameter sweep workflows to map stability regions and edge-case behavior.
  • Implemented coordinate transform and unit-consistency verification tests.
  • Added stress-test scenarios for timing jitter and delayed measurement conditions.
